# 3D-RobotDesign-v2

**Designed for the Smart Methods AI & Robotics Internship**

This repository contains the mechanical CAD files for a fully 3D-printable quadruped robot dog. It was engineered from scratch using Top-Down design methodologies in Onshape, focusing on clean industrial aesthetics and real-world hardware integration.

### 🔗 Live 3D Model
**[[Click Here](https://cad.onshape.com/documents/4dd00338642c7ef631f15e22/w/05f09814197740b1c7d3166c/e/9ceb21190fc7f008eaf759b3)]**

### ⚙️ Hardware Specs
* **Scale:** 600 mm (L) x 350 mm (H)
* **Actuators:** Designed for 12x MG996R standard metal gear servos.
* **Sensors:** V-shaped aerodynamic snout modeled with flush mounts for dual HC-SR04 Ultrasonic Sensors.
* **Kinematics:** Fully articulated leg assemblies split into functional femur and tibia components with built-in 0.5 mm manufacturing tolerances. 

### 🖨️ Manufacturing
The chassis features large planar faces optimized for support-free Desktop FDM 3D printing. All `.STL` files are included in the repository.
